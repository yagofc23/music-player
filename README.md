# ⚡ Music Player - "O Sistema" (Solo Leveling)

> *"ERGA-SE e ouça!"* 🎧

Player de música desktop imersivo inspirado no "Sistema" do anime/manhwa **Solo Leveling**. Desenvolvido inteiramente em Python utilizando as bibliotecas **Tkinter** (para a interface gráfica) e **Pygame** (para a engine de áudio). 

Este projeto foca em reproduzir seus arquivos locais (MP3/WAV) com uma interface "Dark Mode" elegante, trazendo recursos visuais e salvamento de estado para você não perder suas playlists.

---

## ✨ Funcionalidades

*   **Interface Imersiva Dark Mode:** Design escuro e minimalista, inspirado nas telas de status do Monarca das Sombras.
*   **Suporte a Arquivos Locais:** Reproduz arquivos de áudio nos formatos `.mp3` e `.wav`.
*   **Playlists Dinâmicas:** Adicione, remova e gerencie suas músicas facilmente.
*   **Equalizador Gráfico (Visual):** Acompanhamento visual durante a reprodução da música.
*   **Persistência de Dados (Memória do Sistema):** Suas configurações e histórico de playlist são salvos automaticamente no arquivo `system_memory.json`, para que o "Sistema" lembre de tudo ao reiniciar.

---

## 🛠️ Tecnologias Utilizadas

*   **Python 3.x:** Linguagem principal.
*   **Tkinter:** Construção da Interface Gráfica de Usuário (GUI).
*   **Pygame:** Manipulação e reprodução dos arquivos de áudio.
*   **PyInstaller:** Utilizado para empacotar o projeto em um arquivo executável (`.exe`).
